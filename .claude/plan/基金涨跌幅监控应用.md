# 基金涨跌幅监控应用 - 执行计划

## 项目概述
基于微服务架构的基金涨跌幅监控应用，支持数据收集、实时监控、通知和定投回测功能。

## 技术栈
- **后端服务**: Python 3.11 + FastAPI + SQLAlchemy + Pydantic
- **前端框架**: NestJS + Angular + TypeScript + Ant Design
- **数据库**: PostgreSQL + Redis + InfluxDB
- **消息队列**: Redis + Celery
- **容器化**: Docker + Docker Compose

## 核心功能模块
1. **数据收集服务**: 基金信息同步、净值数据采集
2. **监控引擎服务**: 涨跌幅监控、规则引擎
3. **通知服务**: 微信Webhook、邮件通知
4. **回测服务**: 定投策略回测、风险评估
5. **Web管理端**: NestJS + Angular 管理界面
